CSCE 242 Web Applications

Thomas Zwetsloot

This class is part of the University of South Carolina's required CSCE curriculum. Its intent is to teach the skills required to create web applications/websites. Topics covered concerning front end development include HTML and CSS. Later, back end development is taught, first with basic JavaScript and then using Node to enable servers to run js code. There's some JSON thrown in there too, just for fun.
